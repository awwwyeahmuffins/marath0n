## Quick Download Guide:

**BEST FREE SOURCE: Pixabay** (No attribution required, free for commercial use)

1. Go to: https://pixabay.com/gifs/search/
2. Search for each exercise below
3. Download and rename to match the filename
4. Place in this folder

## File Naming Convention:

Name your GIF files exactly as listed below and place them in this folder:

### Lifting Exercises:
- `squat.gif`
- `bench.gif`
- `deadlift.gif`
- `press.gif`
- `rdl.gif`

### Knee Exercises:
- `wall-sit.gif`
- `step-downs.gif`
- `soleus.gif`
- `band-walks.gif`

### Mobility Exercises:
- `couch-stretch.gif`
- `calf-stretch.gif`
- `hamstring-floss.gif`
- `ankle-rocks.gif`
- `hips-9090.gif`

## Optimization:

Before adding GIFs, optimize them using:
- EZGIF: https://ezgif.com/optimize
- Aim for file sizes under 500KB for mobile performance

The app will automatically display these GIFs when available, or show animated SVG placeholders if not found.
