# Quick Start: Get Exercise GIFs in 5 Minutes

## Option 1: Automated Helper (Easiest)
Run the Python helper script:
```bash
cd pwa/exercises
python3 download_helper.py
```
This will open Pixabay search pages for each exercise automatically.

## Option 2: Manual Download

### Step 1: Go to Pixabay
Visit: https://pixabay.com/gifs/search/
